# 個人検証用 AWS アカウントのセットアップ方法

## 1. AWS アカウントの作成

- [AWS 公式サイト](https://aws.amazon.com/)にアクセスし、「Create an AWS Account」をクリックします。
- 必要な情報（メールアドレス、パスワード、アカウント名）を入力し、アカウントを作成します。
- クレジットカード情報を登録します（無料利用枠の範囲内で使用する場合でも必要）。
- 電話番号を入力し、確認コードを受け取って検証します。
- サポートプランを選択します（個人検証用であれば、無料の「Basic Support」で十分です）。

## 2. IAM ユーザーの作成

- AWS マネジメントコンソールにログインし、「IAM」サービスを開きます。
- 「Users」を選択し、「Add user」をクリックします。
- ユーザー名を入力し、アクセスタイプを選択します（個人検証用であれば、「Programmatic access」のみで十分です）。
- 必要な権限を付与するため、適切な IAM ポリシーをアタッチします。
- ユーザーを作成し、アクセスキーとシークレットアクセスキーをダウンロードまたはコピーします。

## 3. MFA の設定（オプション）

- セキュリティを強化するため、多要素認証（MFA）を設定することをお勧めします。
- IAM ユーザーの詳細ページで、「Security credentials」タブを開き、「Assigned MFA device」の「Manage」をクリックします。
- 仮想 MFA デバイスを選択し、スマートフォンの Google Authenticator アプリなどで QR コードをスキャンします。
- 生成された 2 つの認証コードを入力し、MFA を有効化します。
